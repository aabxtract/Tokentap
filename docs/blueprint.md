# **App Name**: TokenTap

## Core Features:

- Wallet Connection: Connect to user's Web3 wallet via the dApp.
- Token Claim: Allow users to claim a small amount of tokens with a single tap.
- Cooldown Timer: Implement a cooldown period between token claims, showing a countdown timer.
- Success Modal: Display a success modal upon successful claim with transaction details.
- Error Handling: Provide user-friendly messages for errors and cooldown states.
- Gas Estimation Tool: Utilize an LLM tool to estimate gas fees and suggest optimal times for claiming to avoid high costs.

## Style Guidelines:

- Primary color: Electric cyan (#7DF9FF) for a futuristic, energetic feel.
- Background color: Deep midnight blue (#191970) with a desaturation of 20% for a dark, calming base.
- Accent color: Neon purple (#9400D3) to highlight interactive elements and create a soft, glowing effect.
- Body and headline font: 'Space Grotesk', a sans-serif font for a geometric, futuristic aesthetic.
- Use a droplet for the app's logo combined with a token shape to represent token distribution. Also, use a lightning bolt for indicating fast transactions.
- Implement glassmorphism panels with rounded corners, soft shadows, and big white spacing for a minimal and clean design.
- Include subtle particle motion in the background, smooth pop-up modals, confetti sparkles, and pulsing claim button animations to create a tactile and responsive user experience.
